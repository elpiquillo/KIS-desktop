
# KIS-FRONT-ADMIN-V2

The new version of kis for admins


## Installation

Use the package manager [npm](https://docs.npmjs.com/try-the-latest-stable-version-of-npm) to install dependencies.

```bash
npm install
```

## Usage

Starting dev

```bash
npm run dev
```
